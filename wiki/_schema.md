# Wiki Schema

## 目录约定

| 目录 | 用途 |
|------|------|
| concepts/ | 公司方法论、原则、判断框架 |
| entities/ | 平台、品类、工具、关键实体 |
| roles/ | 岗位知识、决策树、提问规范 |
| faq/ | 高频问答和标准说法 |
| synthesis/ | 审核通过的综合分析页 |

## 内容规范

- 正式 Wiki 页面只保存审核通过的内容。
- 页面优先表达结论、判断依据、下一步动作。
- 相关页面尽量引用已有页面，避免在多个页面里重复堆相同内容。
- source_ids 优先填写原始 KB 条目编号，方便后续回查。
